Commandes de recherche

/g - Google

/yt - YouTube

/gh - GitHub

/w - Wikipedia

/r - Reddit

/so - Stack Overflow

/i - Google Images

/ddg - DuckDuckGo

/px - Perplexity AI

/a - Amazon

/n - Netflix

/sp - Spotify

/tw - Twitter

/ig - Instagram Tags

/maps - Google Maps

Exemple: /yt chats mignons, recherche directement sur Youtube "chats mignons".

Astuce: Utilise un slash (/) avant une commande, ou bien tape juste ta recherche pour rechercher avec Google
